Detailed Description


Abstract Audio Component, base for every Audio Component.

Base class for every ticking entity.


Abstract class for every light source component.

Component for rigid body, allowing collision to be done.
Base class for every component contained in a TCMSceneObject.


Component for an attenuated light.
